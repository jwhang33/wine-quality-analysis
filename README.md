# Wine Quality Data Analysis

## Overview
Analyzed the Vinho Verde wine dataset to determine which chemical properties are most strongly associated with wine quality.

## Tools
- JMP
- Statistical Analysis

## Dataset
- 1,500+ wine samples

## Objectives
- Explore relationships between chemical measurements and quality ratings
- Apply statistical methods to identify significant variables

## Results
- Identified chemical characteristics that have the strongest relationship with wine quality.
- Collaborated with a team to complete the statistical analysis and presentation.
